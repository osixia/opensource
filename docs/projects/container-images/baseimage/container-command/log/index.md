# Logging Commands

```
Logging utilities

Usage:
  container log [command]

Aliases:
  log

Print Commands:
  fatal       Fatal
  error       Error
  warning     Warning
  info        Info
  debug       Debug
  trace       Trace

Additional Commands:
  level       

Flags:
  -h, --help   help for log

Use "container log [command] --help" for more information about a command.
```
