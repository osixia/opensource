# 🤖 Container Command

```
Usage:
  container [command]

Install Commands:
  install            Install container
  debug              Install debug utilities

Entrypoint Command:
  entrypoint         Container entrypoint

Runtime Commands:
  packages           Packages subcommands
  processes          Processes subcommands
  services           Services subcommands

Environment Commands:
  environment-files  List environment file(s)
  envsubst           Envsubst
  envsubst-templates Envsubst templates

Filesystem Command:
  watch              Watch file(s)

Logger Command:
  logger             Logger subcommands

Additional Commands:
  help               Help about any command
  completion         Generate the autocompletion script for the specified shell

Flags:
  -h, --help   help for container

Use "container [command] --help" for more information about a command.
```
